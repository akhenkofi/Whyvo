# FarmSavior PRD (MVP)

## Product Vision
FarmSavior is a West Africa-focused agri super-platform connecting farmers, buyers, transporters, and service providers with trusted digital workflows.

## Target Regions
- Ghana (GH)
- Nigeria (NG)
- Burkina Faso (BF)

## MVP Objectives
1. Onboard users by role with OTP verification.
2. Provide Digital Farm Passport for farmer identity and production profile.
3. Enable crop trading via listings and offers.
4. Coordinate logistics requests and transporter acceptance.
5. Track mobile money payments with escrow flag.
6. Push weather alerts for farm decisions.
7. Give admins visibility through dashboard metrics.

## Core User Roles
- Farmer
- Buyer
- Transporter
- EquipmentProvider
- StorageProvider
- Admin (platform ops)

## Non-Goals (MVP)
- Full escrow release workflow automation
- Real telco payment integrations
- Real weather provider contracts
- Cross-border legal contracts engine
